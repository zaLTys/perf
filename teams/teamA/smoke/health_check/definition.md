# Health Check Smoke Test

## Purpose
- Quick validation that API is accessible and responding
- Basic functionality check before running larger tests
- Fast feedback on system availability

## Test Type
**Smoke Test** - Minimal load, quick execution

## Expected Outcome
- **RPS**: ~1 req/s (1 VU)
- **Response Time**: p95 < 300ms
- **Success Rate**: > 99%
- **Duration**: ~30 seconds
